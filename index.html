What I'm Learning

Below is a brief overview of the skills I am learning during my time here at Lambda School.

HTML

HTML is the foundation upon which everything else is built. Of course I'm familiar with HTML!

CSS

CSS has the unique feature of being easy to comprehend, but difficult to master. Lambda School prepares you to tackle any layout.

JavaScript

JavaScript powers most of the modern web. I'm learning all of the fundamentals, and the technologies that are built on top of JS.

React

ReactJS is a popular library for building applications.

Redux

Redux is a predictable state management library for JavaScript applications and is the most popular State container that is used for React applications.

Node

Node.js is a runtime environment (a program that runs other programs), a platform used to execute JavaScript applications outside the browser.
